# Inter-VLAN Routing Lab (Router-on-a-Stick)

## Overview
Built a 3-VLAN enterprise network using Cisco routing and switching technologies in GNS3.

## Technologies Used
- Cisco Router
- Cisco Layer 2 Switch
- VLANs
- 802.1Q Trunking
- Router-on-a-Stick
- Inter-VLAN Routing

## VLAN Structure
- VLAN 10 → 192.168.10.0/24
- VLAN 20 → 192.168.20.0/24
- VLAN 30 → 192.168.30.0/24

## Features Implemented
- VLAN segmentation
- Access port assignment
- Trunk configuration
- Router subinterfaces
- Layer 3 communication between VLANs
- Connectivity verification

## Verification
- Successful inter-VLAN ping
- Router reachability
- VLAN verification
- Trunk verification
